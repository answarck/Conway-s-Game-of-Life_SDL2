
# Conway's Game of Life using SDL2

Reference: https://en.wikipedia.org/wiki/Conway's_Game_of_Life



## Dependencies

libsdl2-dev is the only Dependency, To install it,

```bash
sudo apt install libsdl2-dev
```
    
## Running the Simulation

To run just type, 

```
make all
```
